ВНИМАНИЕ! Весь проект должен быть в папке 1210_Birthday_mails. Не меняйте это имя и не меняйте программные файлы!
Для настройки отправителя, текста и темы письма запустите exe\files\settings.exe.
Для проверки наличия именинников и рассылки поздравлений запустите exe\files\start.exe.
В Windows рекомендуем воспользоваться планировщиком задач. А для большего удобства можно создать .bat сценарий, который запустит start.exe
