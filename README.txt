ВНИМАНИЕ! Весь проект должен быть в папке 1210_Birthday_mails. Не меняйте это имя и не меняйте программные файлы!
Для настройки отправителя, текста и темы письма запустите settings.py.
Для проверки наличия именинников и рассылки поздравлений запустите start.py. В файле бесконечный цикл, достаточно запустить при запуске ОС.
В Windows рекомендуем воспользоваться планировщиком задач. А для большего удобства можно создать .bat сценарий, который запустит start.py
